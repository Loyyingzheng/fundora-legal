    What happens to your data

    Personal account data

    • Account credentials and personal profile information
    • Expenses, income, wallets, buckets, bills, and personal goals
    • Cloud backups, backup keys, biometric credentials, and app-linked personal learning data
    • Firebase Authentication access

    Shared Group Event and Group Goal records

    When another linked active member remains, shared history is preserved and your identity becomes Deleted user. Ownership is transferred where required.

    When no other linked active member remains, the shared Event or Goal and its dependent records can be hard deleted, including associated Supabase receipt objects where applicable.

    Processing and recovery

    The page stores only a high-entropy deletion status token in browser session storage so a response-loss or page refresh can recover the deletion status. It does not store your Firebase ID token.
